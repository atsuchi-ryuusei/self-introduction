ガルヒ鍼灸院

製作期間

2025年6月～2025年10月

使用言語

HTML / CSS / JavaScript

使用ツール

Windows 10 / Visual Studio Code / Google Chrome / GitHub / 生成AI(ChatGPT)

リンク

https://atsuchi-ryuusei.github.io/garuhi_harikyuin/

製作背景

・就労移行支援 Webデザインコースの修了課題として制作。

・仕様書およびデザインカンプに基づき、再現性を重視したコーディングを実施。

・Google Fonts、Google Maps、Instagram の埋め込み機能を統合。

・オープンソースのJavaScriptライブラリを導入し、ページ内リンクのスムーススクロール機能を実装。

・モバイルファーストを意識したレスポンシブレイアウトに対応。

課題

・HTMLセマンティクスの誤用：section と article の役割を混同し、構造的な論理性やアクセシビリティに課題を残した。

・工程管理の不徹底：明確な納期を設定しなかったことで、制作期間が長期化。

・命名規則の欠如：クラス名やID名に一貫性がなく、コードの可読性と保守性が低下。

・ブレイクポイント設計：デバイスごとの最適な表示幅（px）の定義に時間を要し、効率的なレスポンシブ設計が不十分。

これらの課題を踏まえ、現在制作しているポートフォリオではセマンティックHTMLの適切な使用や命名規則の整理、制作スケジュールの管理を意識して改善に取り組んでいる。

工夫した点

・複数階層のページ制作が初めてだったため、デザインカンプを確認しながら完成イメージをできる限り再現することを意識して制作。

・スマートフォンで閲覧した際にレイアウトが崩れないよう、レスポンシブ表示を考慮した要素配置を意識して実装。
